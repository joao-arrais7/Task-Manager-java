# 🗂 Gerenciador de Tarefas - Aplicativo de Console Java

## 📌 Sobre o Projeto
Este é um gerenciador de tarefas baseado em console, desenvolvido em Java, aplicando os princípios fundamentais da Programação Orientada a Objetos (POO).

O sistema permite aos usuários:
- Criar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## 🧠 Conceitos Aplicados
- Encapsulamento
- Classes e Objetos
- Listas (ArrayList)
- Lógica condicional
- Tratamento de entrada do usuário (Scanner)
- Organização de código limpa

## 🛠 Tecnologias Utilizadas
- Java
- Eclipse IDE
- Git e GitHub

## 🎯 Objetivo
Este projeto foi desenvolvido como parte da minha jornada de aprendizado em Java, com foco em aprimorar meu entendimento de POO e estrutura de software.
